# Assets folder

Add your logo and other images here. They will be served from `/assets/` (e.g. `/assets/pak_visa_logo.png`).

**Logos used by the app:**

| File | Where it's used |
|------|-----------------|
| `pak_visa_logo.png` | Public visa verification page (`/u/[id]`) — green Pak Visa logo (emblem + “Pak Visa” + “ISLAMIC REPUBLIC OF PAKISTAN”) |
| `header-logo.png` | Site header (if you re-enable it) — Government of Pakistan / Ministry of Interior emblem |

You can add any other images (icons, banners, etc.) in this folder and reference them in your code as `/assets/your-file.png`.
