# Static assets

Static files here are served from the site root. Put logos and images in the **`assets/`** folder:

- **`assets/pak_visa_logo.png`** — public visa verification page (`/u/[id]`). If missing, a text fallback is shown.
- **`assets/header-logo.png`** — for the site header if you add it back.

See **`assets/README.md`** for the full list.
